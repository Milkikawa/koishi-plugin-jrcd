# Changelog

## [0.1.0]

首次发布。

### Added

- `jrcd` 命令（别名 `今日长度`）与 `bbcd` 命令（别名 `比比长度`）。
- 每日结果按平台和用户 ID 生成，当天固定、次日刷新。
- 比较结果文案可配置，区分更大 / 更小两侧的大、中、小差距。
- 回归测试覆盖每日结果、时区、比较边界与参数解析。

### Changed

- 将原单文件逻辑拆分为配置、每日结果、比较、命令模块。
- `bbcd` 改用 Koishi user 参数，修复短于对方时的中等差距判断。

### Fixed

- 修复 `bbcd` 通过全局中间件 `startsWith` 拦截所有消息的问题。原实现绕过 Koishi 命令注册，任何以 `bbcd` 开头的消息都会被误判为命令，未匹配到目标用户时还会直接中断中间件链不调用 `next()`，影响其他插件正常收消息。现改为正式 `bbcd <target:user>` 命令。
- 修复比较时短于对方 11 ～ 20 cm 的文案分支因判断条件写反而无法命中，该区间现在会正确返回对应文案。
- 修复身份键只使用 `userId` 导致不同平台相同 ID 的用户结果碰撞，改为 `platform:userId`。
- 修复日期键依赖 `en-US` locale 格式字符串，在 ICU 版本差异下可能漂移的问题，统一使用 `YYYY-MM-DD`。
- 移除 `String.prototype.hashCode` 全局扩展，该写法在模块加载时污染所有字符串，插件卸载后无法回收。
- 移除每次执行命令时向控制台输出种子、用户 ID 等内部计算材料的调试日志。

### Removed

- 移除未使用的控制台模板与前端产物。

### Notes

- 暂不使用数据库，不保存历史记录。
- 暂无排行榜、定时推送和图片功能。
