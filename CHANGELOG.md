# Changelog

## [0.1.0]

首次发布。

### Added

- `jrcd` 命令（别名 `今日长度`）与 `bbcd` 命令（别名 `比比长度`）。
- 每日结果按平台和用户 ID 生成，当天固定、次日刷新。
- 比较结果文案可配置，区分更大 / 更小两侧的大、中、小差距。
- 回归测试覆盖每日结果、时区、比较边界与参数解析。

### Changed

- 将原单文件逻辑拆分为配置、每日结果、比较、命令模块。
- `bbcd` 改用 Koishi user 参数，修复短于对方时的中等差距判断。

### Removed

- 移除未使用的控制台模板与前端产物。
- 移除 String.prototype 扩展与调试输出。

### Notes

- 暂不使用数据库，不保存历史记录。
- 暂无排行榜、定时推送和图片功能。
